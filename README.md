## 🚀 Quick Start Guide

### 📋 Prerequisites
Make sure you have these installed:
- � **Node.js** (v16 or higher) - [Download here](https://nodejs.org/)
- 📦 **npm** or **yarn** - Package manager
- 🍃 **MongoDB Atlas** - [Get free account](https://www.mongodb.com/atlas)
- 🔥 **Firebase** - [Create project](https://console.firebase.google.com/)
- 💳 **Stripe Account** - [Setup here](https://stripe.com/)

### ⚡ Installation Steps

1️⃣ **Clone the Repository**
```bash
git clone https://github.com/jibon49/desh-explorer-client.git
cd desh-explorer-client
```

2️⃣ **Install Dependencies**
```bash
npm install
# or
yarn install
```

3️⃣ **Environment Setup**
Create a `.env.local` file in the root directory:
```env
# Firebase Configuration
VITE_FIREBASE_API_KEY=your_firebase_api_key
VITE_FIREBASE_AUTH_DOMAIN=your_project.firebaseapp.com
VITE_FIREBASE_PROJECT_ID=your_project_id
VITE_FIREBASE_STORAGE_BUCKET=your_project.appspot.com
VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
VITE_FIREBASE_APP_ID=your_app_id

# Payment Gateway
VITE_STRIPE_PUBLISHABLE_KEY=your_stripe_publishable_key

# API Configuration
VITE_API_BASE_URL=http://localhost:5000
VITE_IMAGE_HOSTING_KEY=your_imagebb_api_key
```

4️⃣ **Start Development Server**
```bash
npm run dev
# or
yarn dev
```

5️⃣ **Build for Production**
```bash
npm run build
# or
yarn build
```
